## الخطوات لإضافة الصور الحقيقية

### 1. إنشاء مجلد للصور

```bash
# في مجلد المشروع
mkdir public\members
mkdir public\news
```

### 2. إضافة صور الأعضاء

ضع صور أعضاء المكتب التنفيذي في:
```
public/members/
  ├── mohamed-ouled-mohamed.jpg
  ├── liwaa-eddine-nabloussi.jpg
  ├── montasser-salem.jpg
  ├── chahine-trabelsi.jpg
  ├── youssef-hosni.jpg
  ├── omaima-ben-ghrib.jpg
  ├── yassine-lagti.jpg
  ├── amanallah-maalaoui.jpg
  ├── yassine-maymoun.jpg
  ├── ahmed-marzougui.jpg
  ├── zied-achek.jpg
  ├── azer-makhloufi.jpg
  ├── maysa-hammadi.jpg
  ├── mohamed-ali-ben-khaled.jpg
  └── mohamed-ihab-saidi.jpg
```

### 3. تحديث روابط الصور في لوحة التحكم

1. افتح لوحة التحكم: `http://localhost:3000/admin`
2. اذهب إلى "المكتب التنفيذي"
3. لكل عضو، حرر واستبدل رابط الصورة:

**مثال:**
```
الصورة القديمة: https://picsum.photos/seed/member1/400/400
الصورة الجديدة: /members/mohamed-ouled-mohamed.jpg
```

### 4. إضافة صورة المؤتمر

1. ضع صورة المؤتمر في: `public/news/congress-2024.jpg`
2. اذهب إلى "الأخبار" في لوحة التحكم
3. حرر خبر "المكتب التنفيذي الجديد"
4. غير رابط الصورة إلى: `/news/congress-2024.jpg`

### 5. مواصفات الصور المثالية

#### صور الأعضاء:
- **الحجم**: 400x400 بكسل (مربع)
- **الصيغة**: JPG أو PNG
- **الحجم**: أقل من 200KB
- **نوع**: صورة شخصية واضحة

#### صور الأخبار:
- **الحجم**: 800x400 بكسل (2:1)
- **الصيغة**: JPG أو PNG
- **الحجم**: أقل من 500KB
- **نوع**: صورة أفقية واضحة

### 6. ضغط الصور (اختياري لكن مستحسن)

استخدم أحد هذه المواقع لضغط الصور:
- https://tinypng.com
- https://squoosh.app
- https://compressor.io

### 7. بعد الانتهاء

```bash
# احفظ التغييرات في Git
git add public/members public/news data/
git commit -m "إضافة صور المكتب التنفيذي الحقيقية"
git push
```

3. الصور تظهر مباشرة

## مثال كامل: تحديث صورة عضو

### الخطوات:

1. **أضف الصورة**:
   - ضع `mohamed-ouled-mohamed.jpg` في `public/members/`

2. **افتح لوحة التحكم**:
   - اذهب إلى `http://localhost:3000/admin`
   - ادخل كلمة المرور: `UGET2026Admin!`

3. **حرر العضو**:
   - اضغط على "المكتب التنفيذي"
   - ابحث عن "محمد أولاد محمد"
   - اضغط على أيقونة التحرير (✏️)

4. **حدث الصورة**:
   - في حقل "رابط الصورة"
   - امسح: `https://picsum.photos/seed/member1/400/400`
   - اكتب: `/members/mohamed-ouled-mohamed.jpg`

5. **احفظ**:
   - اضغط "حفظ"
   - تحقق من الصورة في صفحة المكتب التنفيذي

6. **كرر** للأعضاء الآخرين
